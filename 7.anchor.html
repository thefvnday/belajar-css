Anchor Pseudo-Class

Pendahuluan

Anchor Pseudo-class adalah untuk mendefinisikan status khusus pada elemen anchor/link. Ada beberapa keadaan ketika anchor digunakan. Diantaranya sebagai berikut.

  1. a:link = Keadaan awal dari sebuah link
  2. a:hover= Keadaan dimana sebua link ditunjuk oleh kursor mouse
  3. a:active= Keadaan dimana link sedang aktif
  4. a:visited= Keadaan dimana link sudah pernah dikunjungi
