CASCANDIG STYLE SHEET

Cascanding Style Sheet atau yang biasa kita kenal dengan CSS adalah standar pembuatan dan pemakaian style pada tampilan. Tampilan yang dimaksud disini adalah jenis, ukuran, dan warna font. Selain itu juga jarak antara baris dengan panjangnya, marjin dan indentensi, background, dan warna, serta masih banyak lagi. CSS merupakan fitur yang sangat penting dalam pembuatan Dynamic HTML. Meskipun bukan merupakan kelebihan tersendiri.

Menggunakan CSS tidak diperlukan perangkat lunak terntentu, karena merupakan script yang telah embedded dengan HTML. Hanya dengan menggunakan aplikasi notepad kita dapat menciptakan script CSS sendiri.

Manfaat dari CSS antara lain:

  1. kode HTML menjadi lebih sederhana
  2. ukuran file lebih kecil, sehingga load file lebih cepat
  3. mudah jika ingin mengedit
  4. dapat berkolaborasi dengan javascript
  5. digunakan di hampir semua web browser
